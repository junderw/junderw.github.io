Security Key Series について

Security Key Series (Type-C) は、FIDO認証に特化したYubikeyのエントリーモデルです。シンプルで使いやすく、基本的なセキュリティニーズに最適な選択肢です。上位モデルのYubikey 5シリーズと比較して機能は限定されますが、二要素認証の基本機能を手頃な価格で提供します。

主な機能

  • FIDO2/WebAuthn - パスワードレス認証
  • FIDO U2F - 二要素認証
  • 基本的なセキュリティキー機能
🎯 こんな方におすすめ：
  • 初めてセキュリティキーを使用する方
  • 基本的な二要素認証のみ必要な方
  • コストパフォーマンスを重視する方
  • シンプルな操作を好む方

手順1: NFCチップの有効化

2

USB電源への接続

Security Key SeriesのNFCチップを有効化するには、必ず5秒間USB電源に接続する必要があります。

この手順は以下の場合に必要です：

  • 初回セットアップ時
⚠️ 重要： NFCを使用する前に、必ずUSB-C端子をパソコンやUSB充電器に5秒間接続してください。この手順を忘れると、スマートフォンでNFCが認識されません。
📱 スマートフォンでの使用について：

Security Key SeriesはNFC対応のため、電源接続後はスマートフォンでの無線使用が可能です。Android・iOSの両方でNFC認証をご利用いただけます。

手順2: FIDO PINの設定

1

PINの事前設定が推奨される理由

一部のウェブサイトではFIDO2 PINの作成ができない場合があります。そのため、以下のいずれかの方法で事前にPINを設定することを強く推奨します。

PIN設定方法

方法1: Yubico Authenticatorアプリを使用

  • パソコンにYubico Authenticatorアプリをインストール
  • Security KeyをUSB-C端子に接続
  • アプリでSecurity Keyを認識させる
  • FIDO2設定からPINを設定

方法2: Chromeブラウザの設定を使用

  • Security KeyをパソコンのUSB-C端子に接続
  • Chromeブラウザを開く
  • 設定 → プライバシーとセキュリティ → セキュリティ
  • 「セキュリティキーの管理」を選択
  • PINを設定
💡 ヒント： PINは最低4桁の数字で設定できます。忘れやすいPINは避け、安全で覚えやすいものを選択してください。Security Key SeriesではPINの変更や リセットオプションが限定的なため、慎重に設定してください。

手順3: 基本的な使用方法

3

ウェブサイトでの認証

Security Key Seriesの使用は非常にシンプルです：

基本的な使用手順

  • ステップ1 - ウェブサイトのセキュリティ設定でFIDO2/U2F認証を有効化
  • ステップ2 - Security KeyをパソコンのUSB-C端子に接続
  • ステップ3 - ウェブサイトの指示に従ってSecurity Keyのボタンを押す
  • ステップ4 - 必要に応じてPINを入力

対応サービス例

Security Key Seriesは以下のような主要サービスで使用できます：

  • Google アカウント
  • Microsoft アカウント
  • Facebook / Meta
  • Twitter / X
  • GitHub
  • Dropbox
  • その他FIDO2/U2F対応サービス

管理機能と制限事項

✅ 利用可能な管理機能

  • FIDO2 PINの設定と変更
  • 基本的なセキュリティキー登録と削除
  • デバイス情報の確認

❌ 利用できない機能（Yubikey 5との比較）

  • Yubico OTP - ワンタイムパスワード生成機能
  • 静的パスワード - 固定パスワード入力機能
  • OATH認証 - TOTPトークン管理
  • PIV/スマートカード - 証明書ベース認証
  • OpenPGP - 暗号化・署名機能
  • 高度な設定管理 - 機能トグルなど

HIDデバイスとしての動作について

Security Key SeriesはFIDO認証のみに対応しているため、HIDデバイス（キーボード）としては動作しません。そのため、以下の点にご注意ください：

  • macOSでキーボードアクセス許可を求められることはありません
  • Yubico OTPや静的パスワード機能は使用できません
  • より単純で安全な動作モードとなります

アップグレードを検討するタイミング

以下のような場合は、Yubikey 5シリーズへのアップグレードを検討することをお勧めします：

アップグレードを検討すべき状況

  • TOTP認証アプリの統合 - Google AuthenticatorやAuthyの代替が必要
  • 静的パスワード機能 - 固定パスワードの自動入力が必要
  • 企業環境での証明書認証 - PIV/スマートカード機能が必要
  • 暗号化・署名 - OpenPGP機能が必要
  • より高度なセキュリティ管理 - 詳細な設定制御が必要
